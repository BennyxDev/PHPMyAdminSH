## usage

Syntax:
```
bash <(curl -s https://raw.githubusercontent.com/BennyxDev/PHPMyAdminSH/main/install.sh) [-h|-s]
```
options:

- h  ->   Print the help menu

- s  ->   Save the output in /root/.mariadbPhpma.output
